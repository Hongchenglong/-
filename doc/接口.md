[抽取指定宿舍](http://localhost:8080/cqcq/back_end/public/index.php/index/Draw/customize)

`$parameter = ['department', 'grade', 'block', 'room'];`
注：block和room是**列表**。

如果之前已经被抽到，但这里还可以再指定。

| **KEY** | **VALUE** |
| :-----: | :-------: |
| block[] |   中二    |
| block[] |   东二    |
| room[]  |    411    |
| room[]  |    203    |
